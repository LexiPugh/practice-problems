## The Question

Write a solution to find all the classes that have at least five students.

Return the result table in any order.

<br>

courses Table:

| Column Name | Type    |
|:------------|:--------|
| student     | varchar |
| class       | varchar |

<br>

# My Solution

``` SQL
SELECT
    class
FROM
    courses
GROUP BY
    class
HAVING
    COUNT(student) >= 5
```
